## 一、故事背景（儿童友好版）

### 🚀 欢迎来到零式协约！

你好呀，未来的科学家！

你有没有想过——你桌上那盏小灯是怎么亮的？手机为什么能和你说话？操场上的风为什么会把风筝托上天？

这些问题背后，藏着一个叫**"科学"**的超级力量。而你的大脑，就是操作这股力量的遥控器！

---

### 🌟 什么是"零式协约"？

很久以前，世界是从零开始的。第一团火、第一艘船、第一个飞上天的风筝……所有伟大的发明都有一个共同起点：**零**。

我们把这些爱提问、爱动手的小朋友组成了一支小队，叫**"零式协约"**——

- **"零"** 代表我们从零开始，不怕犯错，每次尝试都是从零到一
- **"式"** 是一种方法、一种风格，属于我们自己的探索方式
- **"协约"** 是大家互相约定的承诺：一起发现、一起动手、一起分享

所以，零式协约就是——**从零出发，一起改变世界的小队！**

---

### 🗺️ 我们的使命

每个队员都是"科创探险家"。我们要做的不是坐在教室里背答案，而是：

1. 🔍 **发现问题** —— 生活中有什么让你好奇的？
2. 💡 **冒出创意** —— 你觉得可以怎么解决它？
3. 🛠️ **动手试试** —— 用纸盒、橡皮筋、乐高……随便什么材料！
4. 🎉 **分享给大家** —— 让全世界知道你的发现！

记住：**没有"笨问题"，只有勇敢的提问者。** 零式协约的每一个队员，都是最棒的！

| **我的超能力** | "你最喜欢做什么？" / "你做得比别人好的事情是什么？" | 不局限于学科，画画好、力气大、会安慰人都算超能力 |

## 三、30秒视频脚本模板

### 📹 统一结构（五镜头法）

每位队员按同一节奏拍摄，后期拼接时风格一致。总时长控制在 **25~30秒**。

| 镜头 | 画面内容 | 台词参考 | 时长 |
|------|---------|---------|------|
| **① 开场自我介绍** | 中景/近景。孩子站定，微笑面对镜头，做一个标志性的手势（比如比心、敬礼、握拳）。 | "大家好！我是零式协约的 `[名字]`！" | ~3秒 |
| **② 发现问题** | 孩子走到一个日常场景前（厨房、阳台、书桌等），表情好奇或疑惑，指着一个东西。 | "你们有没有发现……`[问题]`？" <br>例："为什么家里的垃圾总是分类不对？" | ~6秒 |
| **③ 展示创意** | 孩子展示自己设计的方案——可以是纸模、草图、乐高模型，或对着镜头比划想象中的发明。表情兴奋、有能量。 | "所以我想了个办法！我要做一个`[创意名称]`，它能`[功能]`！" <br>例："我要做一个'垃圾分类小卫士'，它能让垃圾自己找到家！" | ~10秒 |
| **④ 金句宣言** | 特写。孩子直视镜头，认真地说出一句有态度的话。语速放慢，情绪到位。 | "`[个人科创宣言]`" <br>例："科学不是背出来的，是玩出来的！" | ~5秒 |
| **⑤ 统一收尾** | 所有队员用同一画面收束——穿协约T恤（或手举协约标志），一起喊口号。 | "零式协约——从零开始，改变世界！" <br>（齐声） | ~3秒 |

### 🎬 拍摄提示

#### 光线
- **首选自然光：** 白天靠窗拍，脸部均匀受光，避免顶光和逆光
- **备用方案：** 室内用台灯+白纸打补光，不要直接用闪光灯或手机手电筒照脸
- **检查方法：** 拍一张试镜，看孩子眼睛里是否有"眼神光"（两个白色小亮点），有就说明光线OK

#### 背景
- 选择干净、有生活感的场景：书架前、手工桌旁、绿植旁边
- 避免杂乱背景（堆满杂物的房间、电视画面闪烁等）
- 镜头⑤收尾画面统一用一个固定背景，方便后期拼接整齐

#### 拍摄技巧
- **每条多拍2~3遍：** 孩子说错词、表情不到位很正常，保留几条素材方便剪辑选最优
- **声音优先：** 尽量靠近麦克风或使用手机录音模式再配音，比画质更重要
- **不要让孩子背稿：** 把台词变成问题引导出来（"你叫什么名字呀？""你想解决什么问题？"），孩子回答就是最自然的台词
- **横屏拍摄：** 统一横屏16:9，方便后期统一排版

#### 情绪引导
- 开拍前先聊5分钟天，让孩子放松
- 用"我们来玩个游戏"代替"我们要拍视频"
- 孩子说到兴奋处不要打断——即使偏题了也可能是最有感染力的片段

### ✂️ 后期剪辑建议

| 环节 | 建议 |
|------|------|
| **统一片头** | 3秒动画/字幕："零式协约 · 队员档案"，配轻快的开场音效 |
| **转场** | 简单淡入淡出即可，不用炫技特效。保持节奏干净清爽 |
| **背景音乐** | 选用轻快、温暖的纯音乐（无版权风险的lofi或儿童配乐），音量压低到不盖过人声 |
| **字幕** | 每个孩子说台词时配大字号中文字幕，方便家长回看和分享 |
| **队员拼接顺序** | 可随机排列增加新鲜感，每次发布调整一次顺序 |
| **片尾** | 统一logo + 一行标语："零式协约——从零开始，改变世界！" + 日期水印 |
| **导出格式** | MP4，1080P，码率适中。方便微信群/朋友圈直接发送 |
