## 一、项目概述

零式协约是一个**儿童科创小队项目**，核心理念是"从零开始"——孩子们以科创队员的身份，共同学习编程、AI、视频表达和项目协作，最终完成一个可展示的真实作品。

这不是一系列零散的课堂活动，而是一个有明确目标、有组织结构、有迭代过程的真实项目体验。

## 二、核心价值

| 价值维度 | 说明 |
|---------|------|
| **真实项目** | 不是纸上谈兵，最终产出可展示的赛车游戏作品 |
| **团队协作** | 孩子作为队员参与，理解角色分工与协作的意义 |
| **跨学科学习** | 编程 + AI认知 + 视频表达 + 项目管理 |
| **从零到一** | 不假设前置知识，从最小可玩版本开始逐步迭代 |

## 三、项目当前阶段

- **阶段：** 初创期（V0.1）
- **当前核心项目：** 赛车游戏开发
- **第一阶段目标：** 完成最小可玩版本（MVP）

### 最小可玩版本定义：
1. ✅ 赛车能移动
2. ✅ 有赛道
3. ✅ 有起点和终点
4. ✅ 能完成一次完整比赛

**后续迭代方向（不在 V0.1 范围内，仅记录规划）：**
- 障碍物设计
- 碰撞检测
- 计时与排行榜
- 多关卡设计
- 视频展示与作品发布

## 四、项目原则

1. **儿童优先：** 所有内容适合孩子阅读和理解，不使用过度商业化表达
2. **真实成长：** 允许失败和迭代，关注过程而非完美结果
3. **隐私保护：** 不涉及学生联系方式等敏感信息
4. **家长参与：** "天使投资"模式让家长和老师成为项目的支持者而非旁观者

## 五、命名含义

**零式** = 从零开始，不预设前提  
**协约** = 队员之间的约定与协作承诺
