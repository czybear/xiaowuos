# 澄木备课 SOP V0.1

## 一、备课目标
- 保障课堂内容结构清晰、节奏把握精准。
- 形成两套文档：教师备课版（内部使用）与学员投屏版（现场展示）。

## 二、文件命名规范
- `教师备课版.md` → `澄木_教师备课_YYYYMMDD.md`
- `学员投屏版.md` → `澄木_投屏_YYYYMMDD.md`
- 主 SOP 文档固定为 `澄木备课SOP_V0.1.md`

## 三、教师备课版结构
1. **备课目标**
2. **主线提纲**（章节/小节）
3. **完整讲稿**（文字+案例）
4. **互动设计**（提问、演示、作业）
5. **风险提示**（易错点、时间控制）
6. **复盘要点**（学习目标、后续资源）

## 四、学员投屏版结构
- 每页仅一条核心观点
- 大字号标题 + 关键关键词（≤5字）
- 配图/示意（如有）
- 逻辑顺序与教师备课版保持一致

## 五、草帽团队最小分工原则
| 角色 | 任务 |
|------|------|
| **路飞** | 总控调度、结果汇报 |
| **娜美** | 主笔整理全部文档 |
| **索隆** | 最终审核（简洁、可执行、符合断舍离） |
| 其他 | 待命，仅在紧急时提供支援 |

## 六、备课流程
1. **定主线** – 澄木老师确定课程核心目标。
2. **撰写教师版** – 娜美根据提纲完成完整稿。
3. **生成投屏版** – 娜美提炼教师版要点，形成投屏稿。
4. **人工验收** – 索隆快速检查两版是否简洁、符合风格。
5. **归档** – 超出两份的旧稿统一压缩归档，保留最小集合。

## 七、评分标准
- **80 分** – 内容完整，可上线。
- **85 分** – 结构清晰，能动手操作。
- **90 分** – 融入澄木风格（幽默、案例丰富）。
- **95 分** – 可复用模板，形成 SOP 库。

## 八、断舍离归档规则
- 课后 48h 内，仅保留教师版、投屏版两文件。
- 其余草稿、提词卡、旧版本统一压缩至 `archive/YYYYMMDD_澄木_备课_旧版.tar.gz`。

## 九、示例：第 09 课两件套整理
- **教师版**：`澄木_教师备课_20260609.md`
- **投屏版**：`澄木_投屏_20260609.md`
- **归档**：`archive/20260609_澄木_备课_旧版.tar.gz`
